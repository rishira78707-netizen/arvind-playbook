Data Format Guide
The template has 3 sheets — one per BU (Wovens & Knits, Envisol, Advanced Materials). Each sheet has the same structure: Column A = Metric ID (M1–M20), Column B = Metric Name, Column C = Current Value, Column D = Unit. Do not change column order or metric IDs. Leave blank if data is unavailable — missing metrics will be flagged but will not break the upload.

Guide
How to use this dashboard
1
Download & fill the Excel template
Go to the Data Upload section. Download the Arvind HRBP Data Template (.xlsx). It has one sheet with Column A = Metric ID (M1–M20), Column B = Metric Name, Column C = Current Value, Column D = Unit. Fill in your BU's numbers for this month. Leave a cell blank if data is unavailable — it will be flagged but won't break the upload.
2
Upload the file
Drag and drop your completed file into the upload zone, or click Choose File. The dashboard reads the metric columns automatically and populates all 20 metric cards. Any missing or unrecognised columns are flagged in red.
3
Review the Talent Health Score
Once uploaded, check the Talent Health Score — one composite number across 5 clusters. Green = 8–10, Amber = 4–7, Red = 0–3. If any cluster is Red, that is your first conversation point with the business leader.
4
Prepare the CHRO or HR Head view
Switch to CHRO Executive View for the RAG grid — 6 headline metrics, pre-written insight sentences, one recommended action per Red metric. Switch to HR Head View for all 20 metrics with trend lines and cluster-level drill-down.
5
Save a monthly snapshot
After reviewing, scroll to Data Upload → Monthly Snapshot. Label it by month and hit Save Snapshot. Each snapshot builds one point on the 6-month sparkline across all metric cards. Do this every month — the trend line is the story.
6
Log at least one action before you close
Every Red metric must produce one named action in the Action Tracker — with an owner and a deadline. This is non-negotiable. The dashboard is not a report. It is only complete when a decision has been made.
